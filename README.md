# Taxonomy-of-Multimedia-Compression
Multimedia compression taxonomy refers to the systematic classification of compression techniques used for:  Images, Audio, Video, 3D media and Streaming multimedia.  The purpose is to organize compression methods based on:  Data type, Coding principle, Loss characteristics and Redundancy removal techniques
